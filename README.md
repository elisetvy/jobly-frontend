### Live demo
- https://elise-jobly.surge.sh/

### Created with
- JavaScript, React, Express, Tailwind CSS

### Current features
- Can log in/register with JWTs
- Can view company list
- Can view job list
- Can "apply" for a job
- Can search for a company or job with LIVE search
- Can edit profile
